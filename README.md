# Portfolio

Yichen Liu's Journalism Portfolio showcases projects, achievements, and experiences in journalism, graphic design, and software development.

## Features

- **Responsive Design**: The portfolio is designed to be fully responsive and accessible across devices.
- **Interactive Elements**: Includes interactive sections such as a terminal-style resume and project showcases.
- **Custom Styling**: Built with custom CSS and SCSS for a unique and professional look.
- **Dynamic Content**: Uses JavaScript for dynamic content loading and user interaction.

## Directory Structure

```
.
├── about.html          # About page with personal details and achievements
├── assets/             # Contains all static assets (CSS, JS, images, fonts)
│   ├── css/            # Compiled CSS files
│   ├── js/             # JavaScript files for interactivity
│   ├── sass/           # SCSS source files for styling
│   ├── images/         # Image assets
│   ├── webfonts/       # Font files
├── galleries/          # Gallery pages for design, graphics, and photos
├── home.html           # Home page with terminal-style resume
├── index.html          # Main entry point of the portfolio
├── journalism.html     # Journalism portfolio page
├── LICENSE.txt         # License information
├── README.md           # Project documentation
```

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/yliu00/portfolio.git
   ```
2. Open `index.html` in your browser to view the portfolio.

## Technologies Used
- **CSS3/SCSS**: For styling and layout.
- **JavaScript**: For interactivity and dynamic content.
- **FontAwesome**: For icons.
- **Responsive Design**: Ensures compatibility across devices.
